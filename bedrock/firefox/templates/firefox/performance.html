讓 Web 更快

繫好安全帶！最新版 Firefox 擁有閃電般的開機速度、強悍繪圖效能、高速頁面載入等特色，讓你馬上察覺效能明顯提升。

硬體加速

不論你在 Windows 7、Vista、XP 或 Mac OS X 上執行 Firefox，像是看影片或玩遊戲等許多作業都能享受硬體加速的快感。
