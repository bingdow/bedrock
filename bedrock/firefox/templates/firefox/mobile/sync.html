隨時保持同步

完美同步你的 Firefox 瀏覽器，不論何款裝置都能存取自己的資料。

進一步了解 Firefox Sync

對 Firefox Sync 同步功能有疑問？到我們的討論區提出意見。

只要完成下列簡單步驟：

  1. 點選 Firefox 的「工具」，找到「選項」中的「同步」分頁。
    (不知道「選項」視窗在哪嗎？)

  2. 點選「設定 Firefox Sync」，接著「建立新帳號」。

  3. 輸入自己的電子郵件地址，設定密碼並再次輸入以確認該密碼。接著勾選「我同意服務條款與隱私權保護政策」。

  4. 輸入圖片顯示的文字，讓 Firefox 確認你不是機器人。點選「下一步」。

  5. 點選「完成」即可關閉設定視窗，接著回到 Sync 視窗。

  6. 這樣就完成了！Firefox Sync 現在會自動在背景同步你的資料。

保障安全

Firefox Sync 在幕後默默守護你的安全。

只有你看得到：

隨時保護你的資料。只有你能取得自己的密碼與瀏覽記錄等資訊。

安全為上：

Firefox Sync 是唯一具備點對點加密 (End-to-end encryption) 的服務。透過 Sync 的高階安全機制，你可避免有心人士的線上攻擊，或讓公司販售你的資訊。

讓你所有的裝置都能共用個人資訊，並兼顧絕佳的隱私與安全性。
