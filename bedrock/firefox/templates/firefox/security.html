保護個人隱私

在瀏覽網路時，你所留下的資料或多或少都內含個人資訊。Mozilla 認為這些資訊只屬於你個人所有 (某些使用誇張手法的廣告商更不該擁有)，因此 Firefox 內建多項功能以確保個人資訊安全。

{{ platform_img('img/firefox/security/screenshot-privacy.jpg', {'alt': 'Private Browsing screenshot'}) }}

  • 不要追蹤我

    許多網站會追蹤你的線上行為，再將相關資料販售給廣告商。Firefox 能幫你告知所有網站你拒絕此類的追蹤作業。進一步了解。

  • 隱私瀏覽

    有時你就是想私下瀏覽網站，則此功能可保護自己的瀏覽記錄不外洩。進一步了解。

  • 清除最近的歷史記錄

    輕點滑鼠幾下，就能清除所有隱私資料或最近的活動。進一步了解。

  • 刪除網站記錄

    對最近瀏覽過的網站有所疑慮？直接移除所有記錄！進一步了解。

  • 內容安全政策

    Firefox 可讓網站明確指出合法的內容，進而保護使用者免受 XSS 攻擊。

高安全性瀏覽

不論是線上購物、刷卡付賬，或只是登入 Facebook，都應謹慎使用個人資訊以免遭有心人士利用。所幸 Firefox 早已具備高階的安全功能，可時時確保個人資料的安全。

回到頂端

{{ platform_img('img/firefox/security/screenshot-anti-malware.jpg', {'alt': 'Anti-Malware screenshot'}) }}

  • 即時網站識別

    此功能將透過顏色讓使用者區分出可疑網站，避免進入偽造網頁。進一步了解。

  • 反詐騙網頁 & 反惡意軟體

    Firefox 將阻絕木馬與間諜軟體，並警告使用者遠離詐騙網頁。

  • 安全的軟體安裝程序

    必先使用安全連線，才能下載附加元件與其他第三方軟體。

  • 整合防毒程式

    Firefox 完美整合電腦的防毒軟體，讓你能隨時順利下載檔案。

自行控制

為因應變化多端的網路世界，Mozilla 提供簡單易用的工具，讓你能自行判斷並下達聰明的決定。

回到頂端

  • 檢查外掛程式

    過期的第三方外掛程式可能帶來風險，因此 Mozilla 讓你能輕鬆檢查需更新的外掛程式。檢查外掛程式。

  • 家長監護

    強制執行 Windows 7 既有的家長監護功能，停止不適當的下載或網站瀏覽。

  • 客製化安全設定

    多樣的客製化設定，讓你控制 Firefox 對網站訂定的安全層級。進一步了解。

堅守理念

全球性非營利組織 Mozilla 矢志推動網路的平等、自由、開放。我們從未透過使用者而獲利，只想讓大家擁有更完美的上網經驗，也同樣重視個人資訊的安全與隱私。
