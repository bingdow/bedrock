Lightbeam for Firefox

看看誰在線上追蹤你。

揭露誰在線上看著你

Lightbeam 為 Firefox 的附加元件之一，將透過互動式的視覺呈現效果，讓使用者了解自己正在 Web 上產生互動的第一方與第三方網站。當你瀏覽網頁時，Lightbeam 將揭露 Web 串連的廣度與深度，其中亦包含一般使用者平常不會知道的部份。

下載 Lightbeam for Firefox 取自於 Mozilla 附加元件

Lightbeam 運作方式


將 Lightbeam 附加元件下載至 Firefox 瀏覽器並執行之。
針對你瀏覽的所有網站，以及儲存於瀏覽器本端的第三方網站，Lightbeam 均將記錄完整的事件簿。
不論是你自己瀏覽的網站或是第三方網站，Lightbeam 將以圖表方式呈現相關事件，藉以強調這些網站之間所產生的互動。
在你瀏覽 Web 的同時，Lightbeam 亦持續將網站加入該圖表中。你可隨時停用或解除安裝 Lightbeam，也可輕鬆儲存或刪除 Lightbeam 的資料。

追蹤與隱私

我們不需對所有的追蹤作業都抱持負面態度。許多服務都必須依賴使用者的資料，才能提供相關內容並提升線上經驗。但若是在使用者不知情的狀況下執行追蹤作業，就不是每個人都能接受了。如果你願意讓他人共用你的瀏覽資料，也應該由你自己決定共用的時機與方式。Mozilla 了解網路運作透明度的重要性，致力透過工具與相關資訊為使用者提供強大工具。

探索資料庫

在 Lightbeam 資料庫中，將可看到貢獻者透過 Lightbeam 附加元件所上傳的第一方與第三方網站連線資料。Mozilla 正建立此群眾外包 (Crowd-sourced) 資料庫，鼓勵使用者能提供追蹤狀態而統合出整體資料，並檢驗 Web 目前的互動關係。

Web 行為視覺化

為 Firefox 下載並安裝 Lightbeam 附加元件之後，不論是使用者瀏覽中的網站，或是於該網站中活躍的第三方網站，Lightbeam 均能即時呈現網站的相關行為。

分析個人連線

隨著圖表不斷擴大，只要是儲存於自己資料中的多個第一方與第三方網站，你均能進一步了解它們之間的關係。使用者亦可隨時重設或儲存自己的資料。

貢獻開放資料

Mozilla 為 Lightbeam 所規劃的願景之一就是要了解網路追蹤作業，以及第一方與第三方網站的連接方式。如果你願意協助我們，則可將自己的資料貢獻到 Mozilla 的 Lightbeam 資料庫。
